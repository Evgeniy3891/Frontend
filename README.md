# Frontend
*       index.html - стартовый файл
*       <> - открывающий тег
*       </> - закрывающий тег
*       <p> - текст (параграф)
*       <b> - выделение текста жирным шриифтом
*       <i> - выделение текста курсивом
*       <u> - подчеркнутый
*       <ul> - маркированый список
*       <ol> - нумерованый список
*       <img> - вставка картинки
